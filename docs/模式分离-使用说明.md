# fc-tauri 模式分离(Player / Studio)使用说明

> 对应 OpenSpec change `nes-ide-mode-separation`。把原先扁平的 7 视图 + 全局游戏外壳,
> 重构为「模式 → 场景 → 面板」三层架构,玩家与创作彻底分离。

## 三层架构

```
启动 → 模式选择页(Launcher)
        ├─ 🎮 游戏模式(Player)
        │    应用级:游戏库 · 设置
        │    会话级(载入 ROM 才出现):游戏页 + 存档 / 金手指 / 机器级调试(右侧抽屉/覆盖层)
        └─ 🛠 创作模式(Studio)
             IDE 主体(dockview):文件树 · 编辑器 · CHR/地图/音乐 · 构建
             右侧运行列:运行预览(模拟器降为一块面板,可分离浮动)+ 机器检视
             源码级断点调试:断点 ↔ 代码行,命中在编辑器内高亮
```

- 顶层状态 `mode: launcher | player | studio`(`stores/emu.ts`),凌驾于玩家场景 `view` 之上。
- **标题栏常驻模式切换器**:🏠 返回选择页 / 游戏 / 创作,任一处随时互通;记住上次模式
  (`localStorage: fc:lastMode`),选择页高亮上次模式。

## 游戏模式(Player)

- **应用级**:游戏库、设置——任何时候都在。
- **顶部工具栏 = 游戏控制区**(仅游戏激活时显示,见下):游戏页面 · 重置 · 暂停/继续 ·
  快存 · 快读 · **存档 · 金手指 · 调试** · 录像。所有控制与会话入口都在这一处,各自单一入口。
- **会话级**:载入 ROM 即开启一段“会话”。存档管理 / 金手指 / 机器级调试是这局游戏的
  **会话面板**,从顶部工具栏一处入口打开,以**右侧抽屉 / 覆盖层**滑出,游戏留在背景继续运行;
  关闭(× / 点击遮罩 / Esc / 再次点击对应按钮)回到游戏页。
- **底栏 = 应用级导航**:只有 游戏库 · 设置(+ 音量 / 截图)。游戏控制不在底栏。
- **游戏未激活时隐藏控制**:不在游戏页(如游戏库)或没有 ROM 时,顶部所有游戏控制
  (transport + 存档/金手指/调试 + 录像)**整体隐藏**(不是灰掉);仅保留「游戏页面」返回按钮
  (有 ROM 时)。进入游戏模式且无 ROM 时落地**游戏库**。
- **入口去重 / 消歧**(相对旧版与 mockup 的有意改动):
  - 金手指只有工具栏一处入口(删除了右侧面板的「金手指」页签、底栏项)。
  - 「快存 / 快读(槽 1)」(工具栏 + 右侧快捷操作)与「存档管理」(会话抽屉,多槽位/导入导出)
    文案与图标区分开。
  - 删除「停止」按钮(原先只是再暂停,与「暂停」重复);「首页」改名「游戏页面」。
  - “调试”分两类:**机器级**(寄存器/内存/反汇编)是玩家会话面板;**源码级**(断点↔代码行)
    在创作模式。
- **录像工具**:不再是常驻底部录像条;点工具栏「录像」才出现,以**悬浮药丸**浮在底部状态栏带上
  ——不挤压、也不遮挡游戏画面;再点或 × 关闭。
- **运行中药丸**:当游戏在后台被自动暂停(切到游戏库/设置或切去创作)时,标题栏出现
  「运行中:<游戏名>」药丸,点击一键返回游戏页并恢复运行(把旧版隐藏的自动暂停显式化)。

## 创作模式(Studio)

- IDE 是主体;模拟器是右侧运行列里的一块「运行预览」面板,**不再是玩家整页**。
- **构建 → 运行**:把产出 ROM 灌入运行预览并刷新,**停留在 IDE**(不跳玩家页、模式仍 studio)。
- **机器检视**面板(与运行预览同列)复用玩家的机器级检视组件,可与编辑器源码调试同时使用。
- **源码级断点**:在编辑器行号槽打断点,命中暂停时自动切到创作模式并在编辑器高亮对应行
  (沿用 `source-debug-link`)。
- **分离预览**:工具条「分离预览」把预览面板浮动出来便于边改边看,可拖回重新停靠
  (默认停靠;真·独立 OS 窗口留后续增量)。

## 键盘

- 游戏手柄按键(Z/X/方向键/Enter/Space)**仅在游戏模式生效**;在创作/选择页一律交给编辑器与 UI,
  不会被模拟器抢走。

## 主要触点(便于回溯)

`stores/emu.ts`(mode/panel/会话与暂停联动)、`layout/AppShell.vue`(按模式分壳 + 会话抽屉)、
`layout/TitleBar.vue`(模式切换器 + 运行中药丸)、`layout/{Toolbar,FooterNav}.vue`(玩家外壳收口)、
`views/LauncherView.vue`(选择页)、`ide/PreviewPanel.vue`(运行预览)、`views/IdeView.vue`
(预览/检视面板 + doRun)、`composables/useKeyboard.ts`(输入门控)、`composables/useHaltWatch.ts`
(halt → studio)。
